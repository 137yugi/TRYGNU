# 装備デザイン

装備は Diablo 系の「レア度 + 複数アフィックス + ビルドを変える高レア効果」を参照した設計です。レベルアップ能力とは別系統で、同種効果も重複します。

## スロット

装備は2スロットです。

| スロット | 主な役割 |
| --- | --- |
| 本体装備 | HP、移動速度、防御、反射、吸命、回収、XP、ドロップ運、スコア |
| ヌンチャク装備 | 火力、射程、スナップ、ヘッド大型化、回転、衝撃波、連鎖、重力、幻影 |

本体装備とヌンチャク装備は同時に装備できます。`inventory.equipment_slots.body` と `inventory.equipment_slots.nunchaku` に現在装備を出し、既存互換の合算値は `inventory.equipment_mods` に残します。

## レア度

| 色 | 内部ID | 表示 | 基本方針 |
| --- | --- | --- | --- |
| 白 | `common` | コモン | 1アフィックス。序盤の小さな補正 |
| 青 | `magic` | マジック | 2アフィックス。狙いが見える補正 |
| 黄色 | `rare` | レア | 3アフィックス。複数軸を伸ばす |
| 紫 | `epic` | エピック | 4アフィックス。ここから強いビルド変化を含む |
| オレンジ | `legendary` | レジェンダリー | 5アフィックス。専用級の強効果を最低1つ含む |
| 赤 | `ancient` | エンシェント | 6アフィックス。極低確率でゲームバランスを大きく曲げる |

エピック以降は意図的に低確率です。レジェンダリー/エンシェントは通常の数値上昇だけでなく、幻影ヌンチャク、重力井戸、血契、特異点SNAP、赤彗星などの大きな挙動変化を持ちます。

## アフィックス

現行カタログは37種類です。主な系統:

- 火力: ヌンチャク火力、ボス火力、エリート火力、クリティカル、処刑
- 機動: 移動速度、鎖の到達距離、スナップCD、スナップ初速
- 生存: 最大HP、被ダメージ軽減、吸命、接触反撃
- 戦場変化: 衝撃波、連鎖、反射、重力井戸、低HP過給、幻影ヌンチャク
- 報酬: XP獲得、ドロップ運、スコア倍率

装備中の最終補正は `inventory.equipment_mods`、スロット別補正は `inventory.slot_mods` に出ます。レベルアップ能力の `combat.skill_stacks` とは別管理です。

## 実装ルール

- 装備交換時、過去装備の火力加算は残しません。2スロットの `mods` から `equipment_mods` を再計算します。
- 幻影ヌンチャクはスキル由来と装備由来を分けます。ヌンチャク装備を交換しても古い装備由来の幻影は残りません。
- レベルアップ能力と装備アフィックスは別系統で同時に効きます。
- 高レア装備は `pickup_compare` で必ずアフィックス一覧を見てから装備/破棄します。
- 強すぎる効果は許容しますが、キュー詰まり、NaN、装備交換での無限加算は不合格です。
