# 操作取説

## PC

- 移動: `WASD` または 矢印
- メニュー: `M`
- 閉じる: `Esc`
- 選択: `1/2/3` またはクリック。レベルアップ/装備比較はウェーブ全滅後の安全時間で選ぶ
- フルスクリーン: `F`
- デバッグHUD: `H`

## マウス

- Canvasをクリック/ドラッグ: 移動先指定
- 画面左下の開始ボタン: ラン開始/再挑戦

## SP横

- ステージは可視 viewport 全体に表示
- 画面ドラッグ: 指を置いた位置からの相対ドラッグで移動。ドラッグ量が仮想スティック入力になり、離すと停止
- 右上 `メニュー`: 一時停止メニュー
- 右上 `全画面`: 対応ブラウザでは fullscreen 要求

## SP縦

縦画面では内部ステージ自体を縦長に切り替え、可視 viewport 全体を使ってプレイします。下部に `開始/再開`、`メニュー` をオーバーレイします。敵の湧き、広告、ドロップ、ボス予兆も縦長座標で処理されます。
移動はSP横と同じ相対ドラッグです。タップ開始位置へプレイヤーが走り込む挙動にはせず、親指を置いた場所を仮想スティックの中心として扱います。

## ヌンチャク操作

呪鎖ヌンチャクは専用発動ボタンではなく、挑戦者の移動で振り回します。移動方向を変えるとヘッドが慣性で遅れて追従し、テンションがかかったタイミングで弧を描くため、その軌道を敵に重ねます。

## Safari / ホーム画面

通常の Safari タブバー/アドレスバーは Web ページ側から完全には消せません。`manifest.webmanifest` と Apple standalone meta を入れているため、Safari の共有メニューから「ホーム画面に追加」して起動すると、STREAM RAID ARENA のスマホ全画面プレイ向けにブラウザUIを抑えた表示になります。通常ブラウザ内では `visualViewport` の高さへ追従し、タブバー分だけステージが縮んでも操作UIが欠けないようにします。

## 報酬選択

- ウェーブ中: 観客モンスターを全滅させるまで戦闘に集中します。
- ウェーブ全滅後: XPオーブと装備がプレイヤーへ吸い寄せられ、安全時間でレベルアップ3択/装備比較/変異を選びます。
- 選択中: `1/2/3`、クリック、タップで確定します。装備比較は `1` が装備、`2` が破棄です。

## 王者ボス

王者ボスを倒してもランは終了しません。撃破後は報酬回収へ入り、次の王者ボス予定waveが10wave後へ進みます。HPが0になった時点でラン終了です。
