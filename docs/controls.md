# 操作取説

## PC

- 移動: `WASD` または 矢印
- スナップ: `Space`
- メニュー: `M`
- 閉じる: `Esc`
- 選択: `1/2/3` またはクリック。レベルアップ/装備比較はウェーブ全滅後の安全時間で選ぶ
- フルスクリーン: `F`
- デバッグHUD: `H`

## マウス

- Canvasをクリック/ドラッグ: 移動先指定
- `SNAP`: スナップ発動
- 画面左下の開始ボタン: ラン開始/再挑戦

## SP横

- 画面ドラッグ: 移動
- 右下 `SNAP`: スナップ
- 右上 `メニュー`: 一時停止メニュー

## SP縦

縦画面ではゲーム画面を16:9で上部に収め、下部に `開始/再開`、`スナップ`、`メニュー` を表示します。横向き推奨ですが、縦画面でも操作不能にならない設計です。

## 報酬選択

- ウェーブ中: 敵を全滅させるまで戦闘に集中します。
- ウェーブ全滅後: XPオーブと装備がプレイヤーへ吸い寄せられ、安全時間でレベルアップ3択/装備比較/変異を選びます。
- 選択中: `1/2/3`、クリック、タップで確定します。装備比較は `1` が装備、`2` が破棄です。
