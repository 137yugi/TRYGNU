# 操作取説

## PC

- スタート画面: ジョブ、武器、闘士名を選んで `ラン開始`。選択中の闘士画像と武器画像、ロール、難度、HP/速度/火力倍率、初期HP、移動速度、武器込み火力、説明、立ち回り、おすすめ武器が表示されます
- 移動: `WASD` または 矢印
- メニュー: `M`
- 閉じる: `Esc`
- 選択: `1/2/3` またはクリック。レベルアップ/装備比較はウェーブ全滅後の安全時間で選ぶ
- フルスクリーン: `F`
- デバッグHUD: `H`

## マウス

- Canvasをクリック/ドラッグ: 移動先指定
- スタート画面中央の `ラン開始`: ラン開始/再挑戦
- スタート画面中央の `メニュー`: 開始前メニュー。ジョブ/武器の画像付きビルド選択、ジョブ詳細、ラン前ステータス、シーズン、配信イベント、意見フォームへ移動
- ラン中の右下 `STATUS`: 現在HP、速度、武器込み火力、ジョブロール、wave/LV/闘士名を確認

## SP横

- ステージは可視 viewport 全体に表示
- 起動直後はスタート画面からジョブ/武器を選び、`ラン開始` を押す
- ジョブや武器を切り替えると、スタート画面と全画面メニューの闘士/武器画像、ラン中STATUS、ロール/難度/HP/速度/火力/説明/立ち回り/おすすめ武器が即時更新される
- 画面ドラッグ: 指を置いた位置からの相対ドラッグで移動。ドラッグ量が仮想スティック入力になり、離すと停止
- 右上 `メニュー`: 一時停止メニュー
- 右上 `全画面`: 対応ブラウザでは fullscreen 要求
- iPhone Safari の通常タブではブラウザバー分だけ可視領域が変わるため、ステージは `visualViewport` に追従して縮む

## SP縦

縦画面では内部ステージ自体を縦長に切り替え、可視 viewport 全体を使ってプレイします。起動直後は上部にスタート画面、下部に `ラン開始` / `メニュー` を置き、開始後は下部に `開始/再開`、`メニュー` をオーバーレイします。敵の湧き、広告、ドロップ、ボス予兆も縦長座標で処理されます。
移動はSP横と同じ相対ドラッグです。タップ開始位置へプレイヤーが走り込む挙動にはせず、親指を置いた場所を仮想スティックの中心として扱います。

スタート画面の画像付きビルド選択とジョブ詳細は縦長画面では1カラム内に収まり、下部操作デッキと重ならない範囲でスクロールできます。ラン中STATUSは下部デッキの上へ寄せ、長文説明は省略してHP/速度/火力を優先します。

端末を回転すると、次の viewport 更新で縦長/横長ステージを切り替えます。回転直後にSafariのアドレスバーが展開している間は、操作デッキとHUDがsafe-area内に残ることを優先します。

## ヌンチャク操作

呪鎖ヌンチャクは専用発動ボタンではなく、挑戦者の移動で振り回します。停止中は常時モーターのように回り続けず、移動でトルクが入り、方向転換でヘッドが遅れて暴れます。テンションがかかったタイミングで弧を描くため、その軌道を敵に重ねます。

## シーズン自己ベスト

自己ベストはサーバーではなくブラウザの `localStorage` に保存されます。スコア行は `nunchaku_overdrive_scores_v1` に入り、2週間ごとの `seasonId` で分離されます。専用の別キーではなく、同じシーズン内のランキング先頭行が自己ベストです。スタート画面とメニューのシーズン欄は、この値を表示します。

## Safari / ホーム画面

通常の Safari タブバー/アドレスバーは Web ページ側から完全には消せません。`manifest.webmanifest` の `display_override` / `display: fullscreen` と Apple standalone meta を入れているため、Safari の共有メニューから「ホーム画面に追加」して起動すると、STREAM RAID ARENA のスマホ全画面プレイ向けにブラウザUIを抑えた表示になります。通常ブラウザ内では `visualViewport` の高さへ追従し、タブバー分だけステージが縮んでも操作UIが欠けないようにします。

ホーム画面版でもiOSの上端/下端ジェスチャ領域は残るため、ボタンや選択モーダルは `safe-area-inset-*` の内側に置きます。iPadでは縦横どちらも対応しますが、配信や長時間プレイでは横向きまたはホーム画面起動を推奨します。

## 報酬選択

- ウェーブ中: 観客モンスターを全滅させるまで戦闘に集中します。
- ウェーブ全滅後: XPオーブと装備がプレイヤーへ吸い寄せられ、安全時間でレベルアップ3択/装備比較/変異を選びます。
- 選択中: `1/2/3`、クリック、タップで確定します。装備比較は `1` が装備、`2` が破棄です。

## 王者ボス

王者ボスを倒してもランは終了しません。撃破後は報酬回収へ入り、次の王者ボス予定waveが10wave後へ進みます。HPが0になった時点でラン終了です。
