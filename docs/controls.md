# 操作取説

## PC

- 移動: `WASD` または 矢印
- スナップ: `Space`
- メニュー: `M`
- 閉じる: `Esc`
- 選択: `1/2/3` またはクリック。レベルアップ/装備比較はウェーブ全滅後の安全時間で選ぶ
- フルスクリーン: `F`
- デバッグHUD: `H`

## マウス

- Canvasをクリック/ドラッグ: 移動先指定
- `SNAP`: スナップ発動
- 画面左下の開始ボタン: ラン開始/再挑戦

## SP横

- ステージは可視 viewport 全体に表示
- 画面ドラッグ: 移動
- 右下 `SNAP`: スナップ
- 右上 `メニュー`: 一時停止メニュー
- 右上 `全画面`: 対応ブラウザでは fullscreen 要求

## SP縦

縦画面でもステージを可視 viewport 全体に表示します。下部に `開始/再開`、`スナップ`、`メニュー` をオーバーレイします。横向き推奨ですが、縦画面でも操作不能にならない設計です。

## Safari / ホーム画面

通常の Safari タブバー/アドレスバーは Web ページ側から完全には消せません。`manifest.webmanifest` と Apple standalone meta を入れているため、PWAとしてホーム画面に追加して起動すると、神経電脈のスマホ全画面プレイ向けにブラウザUIを抑えた表示になります。通常ブラウザ内では `visualViewport` の高さへ追従し、タブバー分だけステージが縮んでも操作UIが欠けないようにします。

## 報酬選択

- ウェーブ中: ノイズ群を全滅させるまで戦闘に集中します。
- ウェーブ全滅後: XPオーブと装備がプレイヤーへ吸い寄せられ、安全時間でレベルアップ3択/装備比較/変異を選びます。
- 選択中: `1/2/3`、クリック、タップで確定します。装備比較は `1` が装備、`2` が破棄です。

## 過負荷中枢

過負荷中枢を倒してもランは終了しません。撃破後は報酬回収へ入り、次の過負荷中枢予定waveが10wave後へ進みます。HPが0になった時点でラン終了です。
